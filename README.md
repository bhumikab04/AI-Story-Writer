# AI Story Writer

A Streamlit app that turns a short story idea into a full tale using Google Gemini, with optional text-to-speech playback.

## Setup

1. Create a virtual environment and install dependencies:

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Copy `.env.example` to `.env` and add your Google API key:

```bash
cp .env.example .env
```

3. Run the app:

```bash
streamlit run app.py
```

## Features

- Generate stories by genre and length
- Download stories as text files
- Listen to stories with text-to-speech
